createModel

  • createModel(dataModelParams, options?): DataModel | SDKError

  • Creates a new @xeokit/data!DataModel in this Data.

    Remember to call DataModel.build when you've finished building or loading the DataModel. That will fire events via Data.onModelCreated and DataModel.onBuilt, to indicate to any subscribers that the DataModel is built and ready for use.

    Note that while we're building/loading the SceneModel, each call that we make to DataModel.createObject will create a new @xeokit/data!DataObject in Data.objects and DataModel.objects, and will also fire an event via Data.onObjectCreated. However, only when we've received the Data.onModelCreated and DataModel.onBuilt events can we actually consider the DataModel to be fully constructed.

    See @xeokit/data for more details on usage.

    Parameters

    • dataModelParams: DataModelParams

      Creation parameters for the new @xeokit/data!DataModel.

    • Optional options: {
          excludeRelating?: string[];
          includeRelating?: string[];
      }

      Options for creating the @xeokit/data!DataModel.

      • Optional excludeRelating?: string[]

        When provided, never create DataObjects with types in this list.

      • Optional includeRelating?: string[]

    Returns DataModel | SDKError

    @xeokit/data!DataModel
