Class DirLight

A directional light source within a @xeokit/viewer!View.

Summary

  • Illuminates all objects equally from a given direction.
  • Has an emission direction vector in DirLight.dir, but no position.
  • Defined in either World or View coordinate space. When in World-space, DirLight.dir is relative to the World coordinate system, and will appear to move as the @xeokit/viewer!Camera moves. When in View-space, DirLight.dir is relative to the View coordinate system, and will behave as if fixed to the viewer's head.
  • AmbientLights, DirLights and PointLights are registered by their Component.id on View.lights.
