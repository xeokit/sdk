Class FrustumProjection

FrustumProjection-based perspective projection configuration for a @xeokit/viewer!Camera .

  • Located at Camera.frustumProjection.
  • Allows to explicitly set the positions of the left, right, top, bottom, near and far planes, which is useful for asymmetrical view volumes, such as for stereo viewing.
  • FrustumProjection.near and FrustumProjection.far specify the distances to the clipping planes.
  • FrustumProjection.onProjMatrix will fire an event whenever FrustumProjection.projMatrix updates, which indicates that one or more other properties have updated.
