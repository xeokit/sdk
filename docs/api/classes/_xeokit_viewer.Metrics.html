Class Metrics

Configures its @xeokit/viewer!View's measurement unit and mapping between the Real-space and World-space 3D Cartesian coordinate systems.

Summary

  • Located at View.metrics.
  • Metrics.units configures the Real-space unit type, which is @xeokit/constants!MetersUnit by default.
  • Metrics.scale configures the number of Real-space units represented by each unit within the World-space 3D coordinate system. This is 1.0 by default.
  • Metrics.origin configures the 3D Real-space origin, in current Real-space units, at which this @xeokit/viewer!View's World-space coordinate origin sits, This is [0,0,0] by default.

Usage

import {Viewer, constants} from "xeokit-viewer.es.js";

const viewer = new Viewer();

const view1 = myViewer.createView({
     id: "myView",
     elementId: "myCanvas1"
});

const metrics = view1.metrics;

metrics.units = MetersUnit;
metrics.scale = 10.0;
metrics.origin = [100.0, 0.0, 200.0];
