Function loadCityJSON

  • loadCityJSON(params, options?): Promise<any>

  • Loads CityJSON into a SceneModel and/or a DataModel.

    • Expects SceneModel.built and SceneModel.destroyed to be false
    • Does not call SceneModel.build - we call that ourselves, when we have finished building the SceneModel

    See @xeokit/cityjson for usage.

    Parameters

    • params: {
          dataModel?: DataModel;
          fileData: any;
          sceneModel: SceneModel;
      }

      Loading parameters.

      • Optional dataModel?: DataModel

        DataModel to load into.

      • fileData: any

        CityJSON file data.

      • sceneModel: SceneModel

        SceneModel to load into.

    • options: {
          rotateX?: boolean;
      } = ...

      CityJSON loading options

      • Optional rotateX?: boolean

        True to rotate the model about the X-axis. Default is false.

    Returns Promise<any>

    Resolves when CityJSON has been loaded into the SceneModel and/or DataModel.

    Throws

    SDKError

    • If the SceneModel has already been destroyed.

    • If the DataModel has already been destroyed.
