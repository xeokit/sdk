Interface RendererObjectInternal

Interface through which a ViewObject controls the appearance of a SceneObject in a Viewer.

When a Scene is attached to a Viewer, the Viewer attaches a RendererObject to each of the Scene's SceneObjects, to provide an interface through which the Viewer's ViewObject can control the appearance of the SceneObject within a View.

Internally, the Viewer's @xeokit/viewer!Renderer attaches these to SceneObject.rendererObject and @xeokit/viewer!ViewObject.rendererObject | ViewObject.rendererObject. When we update properties like ViewObject.visible, this interface will upload those updates into the @xeokit/viewer!Renderer.
