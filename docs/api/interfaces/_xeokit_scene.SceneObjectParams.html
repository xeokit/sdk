Optional layerId

layerId?: string

Causes each @xeokit/viewer!View to put the corresponding @xeokit/viewer!ViewObject into a @xeokit/viewer!ViewLayer with this ID.

When you create an object in a model, each @xeokit/viewer!View will automatically create a @xeokit/viewer!ViewObject to represent it. When the object has a @xeokit/scene!SceneObjectParams.layerId property, then each View will first ensure that it has a @xeokit/viewer!ViewLayer with this ID, and will register the ViewObject in that ViewLayer.
