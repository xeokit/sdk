Module @xeokit/dtx - v0.1.0

xeokit DTX Format Importer and Exporter

Import and export SceneModels as xeokit's native binary DTX format

This package allows us to import and export xeokit SceneModels as DTX, a compact binary-encoded runtime asset delivery format for geometry and materials.

To import a DTX model into xeokit, use the loadDTX function, which will load the file into a SceneModel. To export a DTX model, use the saveDTX function, which will save a SceneModel to DTX.



Installation

npm install @xeokit/dtx

Usage

In the example below, we'll use loadDTX to import a DTX file into a SceneModel. The SDKError class is used to handle errors that may occur during the process:

import {Scene} from "@xeokit/scene";
import {loadDTX, saveDTX} from "@xeokit/dtx";

const scene = new Scene();

const sceneModel = scene.createModel({
    id: "myModel
});

if (sceneModel instanceof SDKError) {
     console.error(dataModel.message);
} else {
     fetch("myModel.dtx").then(response => {
        response.arrayBuffer().then(fileData => {
             loadDTX({ fileData, sceneModel });
             sceneModel.build();
         });
     });
});

Using saveDTX to export the SceneModel back to DTX:

const arrayBuffer = saveDTX({
    dataModel,
    sceneModel
});
