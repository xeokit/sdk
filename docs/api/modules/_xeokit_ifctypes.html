Module @xeokit/ifctypes - v0.1.0

xeokit IFC Data Types

  • Defines numeric constants for IFC entity and relationship types.
  • Use with @xeokit/data to assign IFC types to DataObjects and Relationships and treat them as IFC elements.
  • Use with @xeokit/treeview to configure the appearance and behaviour of TreeViews for navigating IFC element hierachies.
  • Supports IFC versions 2x3 and 4.

Installation

npm install @xeokit/basictypes
