Module @xeokit/treeview - v0.1.0


xeokit Tree View UI

  • @xeokit/treeview!TreeView
  • Fast HTML tree view to navigate federated models in a Viewer
  • Use with a View and a semantic Data model
  • Supports Industry Foundation Classes (IFC)
  • Supports any schema expressable as ER graph with aggregation relationships

Installation

npm install @xeokit/treeview
