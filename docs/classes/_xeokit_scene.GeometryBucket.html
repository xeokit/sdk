positionsCompressed

positionsCompressed: IntArrayParam

GeometryBucketHandle bucket's 3D vertex positions, quantized as 16-bit integers.

Internally, the Viewer uses positionsDecompressMatrix to decompress (dequantize) these back to 32-bit floating-point relative-to-center (RTC) positions that are relative to origin.

Vertex positions are required for all primitive types.

  • Defined in scene/src/GeometryBucket.ts:22
