Class Frustum

Frustum-based perspective projection configuration for a Camera .

  • Located at frustum.
  • Allows to explicitly set the positions of the left, right, top, bottom, near and far planes, which is useful for asymmetrical view volumes, such as for stereo viewing.
  • near and far specify the distances to the clipping planes.
  • onProjMatrix will fire an event whenever projMatrix updates, which indicates that one or more other properties have updated.
