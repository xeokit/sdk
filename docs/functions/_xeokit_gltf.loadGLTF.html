Function loadGLTF

  • loadGLTF(params: {
        data: ArrayBuffer;
        dataModel?: DataModel;
        log?: Function;
        sceneModel?: SceneModel;
    }): Promise<any>

  • Loads glTF file data from an ArrayBuffer into a SceneModel and/or a DataModel.

    • Expects SceneModel.built and SceneModel.destroyed to be false
    • Does not call SceneModel.build - we call that ourselves, when we have finished building the SceneModel

    See @xeokit/gltf for usage.

    Returns

    Resolves when glTF has been loaded.
