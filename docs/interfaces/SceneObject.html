Interface SceneObject

GeometryBucketHandle and material for an object within a Viewer.

Overview
  • Belongs to a SceneModel, which belongs to a Scene
  • Created with SceneModel.createObject
  • Registered by SceneModel.id in {@link SceneModel.sceneObjects} and {@link Scene.sceneObjects}
  • Has a corresponding ViewObject in each of the Viewer's Views
  • Can have a corresponding DataObject in the Viewer's Data
