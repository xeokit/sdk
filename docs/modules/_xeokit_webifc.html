Module @xeokit/webifc

xeokit IFC Importer
  • Industry Foundation Classes (IFC) is a data model for building information modeling (BIM) used in the architecture, engineering, and construction (AEC) industry.
  • Use loadWebIFC to load IFC files into SceneModels and DataModels.
  • Internally, loadWebIFC uses the web-ifc API to parse geometry and data from the IFC file.

Installation

 
npm install @xeokit/webifc

Usage

Loading an IFC file into a SceneModel and a DataModel:

import {Scene} from "@xeokit/scene";
import {Data} from "@xeokit/data";
import {loadWebIFC} from "@xeokit/webifc;
import * as WebIFC from "web-ifc/web-ifc-api-node";

const scene = new Scene();

const sceneModel = scene.createModel({
    id: "myModel"
});

const data = new Data();

const dataModel = data.createModel({
     id: "myModel"
});

const ifcAPI = new WebIFC.IfcAPI();

ifcAPI.SetWasmPath("path/to/wasm");

ifcAPI.Init().then(() => {

    fs.readFile("./tests/assets/IfcOpenHouse4.ifc", (err, fileData) => {

         if (err) {
            console.error(err);
            return;
         }

         loadWebIFC({
             data: toArrayBuffer(fileData),
             ifcAPI,
             sceneModel,
             dataModel
         }).then(() => {

             sceneModel.build();
             dataModel.build();

         }).catch((e) => {
             console.error(e);
         });
     });
});
